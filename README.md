# Lung Nodule Detection
Using a 3D Vision Transformer (ViT) to detect lung nodules from CT images through end-to-end training. The exmaple images below show patches from [Luna16](https://luna16.grand-challenge.org/Home/), each with a detected nodule highlighted in a 3D bounding box predicted by the model.
